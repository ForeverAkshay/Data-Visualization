# UK Housing Energy Efficiency & CO₂ Emissions Dashboard

A Power BI dashboard exploring the relationship between property type, 
location, energy efficiency, and CO₂ emissions across UK regions, 
built using EPC (Energy Performance Certificate) data.

## Overview
This project visualises:
- **Location task**: Regional spread of CO₂ emissions by property type 
  (clustered column chart)
- **Time task**: Evolution of energy efficiency over time, filterable 
  by region (line chart with slicer)
- **Multi-dimensional task**: Correlation between property type, region, 
  CO₂ emissions and time period (bubble/scatter chart)

## Design principles applied
- Colour-blind-friendly palette (tested for deuteranopia)
- Consistent visual hierarchy, alignment, and spacing
- Interactive slicers (region, time period) and dynamic KPIs
- Minimal, focused use of text and legends
